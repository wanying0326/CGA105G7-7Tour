<h3>技術</h3>
<ul>
<li>程式語言 ： Java17 + JavaSript</li>
<li>資料庫 ： MySQL 8.0 + Redis 3.2</li>
<li>MVC架構 ： JDBC + JSP + Servlet 4 + Tomcat 9</li>
<li>補充 ： JQuery、Ajax、WebSocket、Ecpay、Bootstrap5</li>
<li>IDE：Eclipse 2022-03 (4.23)</li>
</ul>
